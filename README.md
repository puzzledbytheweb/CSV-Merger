# CSV-Merger

CSV-Merger is a library to merge many CSV files into a bigger one

## Installation

```bash
npm install csv-merger
```

Or, if you're using Yarn

```bash
yarn add csv-merger
```

## Usage

```javascript
const csvMerger = require("csv-merger");
```
